# 💡 How It Works

## 1️⃣ Upload Resume
Upload your resume in PDF or DOCX format.

## 2️⃣ Analyze Resume
The system analyzes:
- ATS Score
- Skills
- Keywords
- Resume Structure

## 3️⃣ AI Suggestions
Get AI-powered recommendations to improve your resume.

## 4️⃣ Download Optimized Resume
Export your updated resume in professional format.

# ⚙️ Setup Instructions

## 1️⃣ Clone Repository

```bash
git clone https://github.com/piyushkumar2442/resume-screening-system.git
cd resume-screening-system
2️⃣ Create Virtual Environment
python -m venv venv
Activate Environment
Windows
venv\Scripts\activate
Linux / Mac
source venv/bin/activate
3️⃣ Install Dependencies
pip install -r requirements.txt
4️⃣ Download spaCy Model
python -m spacy download en_core_web_sm
5️⃣ Create .env File

Inside utils folder create:

GOOGLE_API_KEY=your_google_gemini_api_key
6️⃣ Run Application
streamlit run app.py
